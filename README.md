# contactos

## Project setup
```
npm install
```
### instalar AXIOS
```
npm install axios
```
### instalar json SERVER de manera global 
```
npm install -g json-server
```
### Compiles and hot-reloads for development
```
npm run serve
```
### Inicia el servidor JSON
```
json-server --watch db.json --port 3000
```
### Compiles and minifies for production
```
npm run build
```
